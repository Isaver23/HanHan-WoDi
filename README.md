# HanHan-WoDi
## 第一步
### 需要准备好node环境  官网:http://nodejs.cn/
## 第二步
### 使用git将代码clone到本地
` git clone git@github.com:hanhan-GKD/HanHan-WoDi.git `
### 不会git 可在此网站下载zip包，地址:
`https://github.com/hanhan-GKD/HanHan-WoDi/archive/master.zip`

## 第三步
### 注意config中index.json配置,server为部署的服务器ip地址,client则为要连接的服务器ip地址
### 配置好config/index.json后根目录
`node client/main.js 即可连接服务器`
## 第四步
### 根据游戏内相关提示游玩即可
